# FixTar E-Commerce — Image Directory Structure

## /logo
Brand logos in all variants.
- `fixtar-logo-primary.webp` — Main teal logo (use on light backgrounds)
- `fixtar-logo-white.webp` — White logo (use on dark backgrounds, footers, banners)

## /hero
Homepage hero banners and full-width section images.
- Recommended size: 1920×800px
- Use: homepage slider, seasonal promotions, sale events

## /banners
Promotional banners for categories, sales, and campaigns.
- Recommended size: 1200×400px (full-width), 600×400px (half-width)
- Use: category headers, promo strips, email headers

## /products
Product photography, organized by category:
- `/power-tools` — Drills, saws, grinders, sanders, etc.
- `/hand-tools` — Wrenches, screwdrivers, pliers, hammers, etc.
- `/accessories` — Drill bits, blades, chargers, batteries, etc.
- `/safety-equipment` — Gloves, goggles, ear protection, masks, etc.

### Product image naming convention:
`[brand]-[product-name]-[variant]-[view].webp`
Example: `bosch-impact-drill-blue-front.webp`

Views: front, side, top, detail, lifestyle, packaging

## /categories
Category thumbnail/card images (one per product category).
- Recommended size: 800×800px or 800×600px
- Naming: `cat-[category-name].webp`

## /about
Team photos, warehouse, storefront, brand story images.

## /icons
UI icons, feature icons, trust badges, payment icons.
- Preferred format: SVG or small WebP
- Naming: `icon-[name].svg`
